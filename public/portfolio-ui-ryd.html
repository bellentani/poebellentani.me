RYD Agro

A RYD é uma startup de Belém/PA com foco em leilões de ofertas de compra de comodities agrícolas. Fiz a Direção de Arte para desenvolvimento de logomarca e, em seguida, wireframes e fluxo navegacional, entregando um protótipo navegável para ser implementado pela equipe de desenvolvimento, juntamente com um UI KIT baseado em Google Material.

Portfolio UI UX

UI Guide

Como o requisito principal era acelerar o desenvolvimento do aplicativo, escolhemos o Material Design como base - personalizei a interface e apliquei modificações para ficar dentro do guia de utilização da marca.

Design do App

O aplicativo ganhou uma direção de arte baseada no UI Guide. Foi trabalhada a jornada dos usuários e os dois grandes grupos de personas: fluxos foram mapeados, o trabalho de UX desenvolvido e um protótipo entregue.

Direção de Arte

Não só fizemos a parte online, mas criamos um guia de marca. O logotipo, inclusive, foi trabalhado pensando em aplicação digital e em como iria interagir juntamente com ícones de aplicativos, sites e identidades digitais.

UI Guide e Guia da marca

Realizei levantamento de briefing junto ao cliente, estudos de marcas, anseios e trabalho de direção de arte junto a designer que desenvolveu o conceito do logo. O meu trabalho foi traduzir os anseios do cliente para que fosse desenvolvida uma marca coerente e aplicável em meios digitais.
